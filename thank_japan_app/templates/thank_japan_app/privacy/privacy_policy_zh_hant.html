隱私權政策

ThankJapan（以下簡稱「本公司」）致力於保護您的隱私。我們將根據《個人資料保護法》及相關法律規定處理您的個人資訊。 本隱私權政策說明了當您使用我們的網站、服務及遊戲時，我們如何處理您的個人資料。

1. 我們收集的資訊

  • 帳號資訊： 我們收集您的使用者名稱和密碼的雜湊（加密）版本。我們不會以明文形式儲存密碼。
  • 遊戲數據： 我們收集遊戲分數和進度，以提供功能、排行榜並增強您的體驗。
  • 付款數據： 付款處理由 {% if is_twa %}Google Play{% else %}PayPal{% endif %} 安全處理。我們不儲存信用卡卡號。我們僅儲存由 {% if is_twa %}Google{% else %}PayPal{% endif %} 提供的訂閱 ID 和付款狀態，以管理您的存取權限。
  • AI 隱私： 我們不會將您的個人數據、使用者名稱或分數提供給外部人工智慧模型進行訓練。

2. 我們如何使用資訊

  • 管理您的帳號並提供高級方案的存取權限。
  • 透過 {% if is_twa %}Google Play{% else %}PayPal{% endif %} 安全地處理每月訂閱。
  • 在排行榜上顯示分數並儲存您的學習進度。
  • 確保我們平台的安全性和穩定性。

3. Cookie 與安全性

我們使用對於安全登入、維持工作階段以及確保 {% if is_twa %}付款{% else %}PayPal{% endif %} 交易完整性至關重要的 Cookie。

4. 資料共享與第三方

我們僅與營運所需的信任夥伴共享資料：

  • {% if is_twa %}Google Play{% else %}PayPal{% endif %}： 用於付款處理和訂閱管理。
  • Cloudinary： 用於提供優化的教育圖像。
  • Google Analytics： 用於匿名使用情況分析。

5. 用戶權利與數據刪除

您有權隨時訪問、更正或刪除您的數據。您可以透過「帳戶設定」頁面立即永久刪除您的帳戶及所有相關歷史記錄。一旦刪除，數據將無法恢復。

6. AI 技術的使用

我們利用 AI 技術生成某些圖像和教學內容。雖然我們追求高準確性，但這些內容僅供學習輔助之用，可能存在 AI 技術固有的局限性。

7. 國際數據傳輸

使用我們的服務即表示您同意將您的數據在位於美國或日本等司法管轄區的服務器上進行處理，我們的基礎設施託管於此。

10. 帳號刪除
