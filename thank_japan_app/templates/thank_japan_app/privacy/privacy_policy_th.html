นโยบายความเป็นส่วนตัว

ThankJapan ("บริษัท") มุ่งมั่นที่จะปกป้องความเป็นส่วนตัวของคุณ เราดำเนินการประมวลผลข้อมูลส่วนบุคคลของคุณตามพระราชบัญญัติคุ้มครองข้อมูลส่วนบุคคล (PDPA) ของประเทศไทย

1. ข้อมูลที่เราเก็บรวบรวม

  • ข้อมูลบัญชี: เราเก็บรวบรวมชื่อผู้ใช้และรหัสผ่านในรูปแบบที่เข้ารหัส (Hashed) เราไม่เก็บรหัสผ่านเป็นข้อความธรรมดา
  • ข้อมูลเกม: เราเก็บรวบรวมคะแนนเกมและความคืบหน้าเพื่อให้บริการฟังก์ชันต่างๆ ตารางอันดับ และเพิ่มประสบการณ์การใช้งานของคุณ
  • ข้อมูลการชำระเงิน: การประมวลผลการชำระเงินดำเนินการอย่างปลอดภัยโดย {% if is_twa %}Google Play{% else %}PayPal{% endif %} เราไม่เก็บข้อมูลบัตรเครดิต เราเก็บเพียง ID การสมัครสมาชิก และสถานะการชำระเงินที่ได้รับจาก {% if is_twa %}Google{% else %}PayPal{% endif %} เพื่อจัดการการเข้าถึงของคุณ
  • ความเป็นส่วนตัวด้าน AI: เราจะไม่ให้ข้อมูลส่วนบุคคล ชื่อผู้ใช้ หรือคะแนนของคุณแก่โมเดลปัญญาประดิษฐ์ภายนอกเพื่อการฝึกฝน

2. วิธีที่เราใช้ข้อมูล

  • เพื่อจัดการบัญชีของคุณและให้สิทธิ์เข้าถึงแพ็กเกจพรีเมียม
  • เพื่อประมวลผลการสมัครสมาชิกรายเดือนอย่างปลอดภัยผ่าน {% if is_twa %}Google Play{% else %}PayPal{% endif %}
  • เพื่อแสดงคะแนนในตารางอันดับและบันทึกความคืบหน้าการเรียนรู้ของคุณ
  • เพื่อรักษาความปลอดภัยและความเสถียรของแพลตฟอร์มของเรา

3. คุกกี้และความปลอดภัย

เราใช้คุกกี้ที่จำเป็นสำหรับการเข้าสู่ระบบที่ปลอดภัย การคงเซสชัน และเพื่อรับรองความสมบูรณ์ของธุรกรรม{% if is_twa %}การชำระเงิน{% else %} PayPal{% endif %}

4. การแชร์ข้อมูลและบุคคลที่สาม

เราแชร์ข้อมูลกับพันธมิตรที่เชื่อถือได้ที่จำเป็นต่อการดำเนินงานเท่านั้น:

  • {% if is_twa %}Google Play{% else %}PayPal{% endif %}: สำหรับการประมวลผลการชำระเงินและการจัดการการสมัครสมาชิก
  • Cloudinary: สำหรับการจัดส่งรูปภาพเพื่อการศึกษาที่ได้รับการปรับแต่ง
  • Google Analytics: สำหรับการวิเคราะห์การใช้งานแบบไม่ระบุตัวตน

5. สิทธิ์ของผู้ใช้และการลบข้อมูล

คุณมีสิทธิ์ในการเข้าถึง แก้ไข หรือลบข้อมูลของคุณได้ทุกเมื่อ คุณสามารถ ลบบัญชีและประวัติทั้งหมดที่เกี่ยวข้องอย่างถาวร ได้ทันทีผ่านหน้า "การตั้งค่าบัญชี" (Account Settings) เมื่อลบแล้ว ข้อมูลนี้จะไม่สามารถกู้คืนได้

6. การใช้เทคโนโลยี AI

เราใช้เทคโนโลยี AI เพื่อสร้างรูปภาพและเนื้อหาเพื่อการศึกษาบางส่วน แม้ว่าเราจะมุ่งเน้นความถูกต้องแม่นยำสูง แต่ข้อมูลเหล่านี้จัดทำขึ้นเพื่อสนับสนุนการศึกษาและอาจมีข้อจำกัดที่มาพร้อมกับเทคโนโลยี AI

7. การโอนข้อมูลระหว่างประเทศ

ในการใช้บริการของเรา คุณยินยอมให้ข้อมูลของคุณถูกประมวลผลบนเซิร์ฟเวอร์ที่ตั้งอยู่ในเขตอำนาจศาล เช่น สหรัฐอเมริกาหรือญี่ปุ่น ซึ่งเป็นที่ตั้งของโครงสร้างพื้นฐานของเรา
