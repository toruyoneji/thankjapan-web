隐私政策

ThankJapan（以下简称“本公司”）致力于保护您的隐私。我们将严格遵守《中华人民共和国个人信息保护法》及相关法律法规。 本隐私政策旨在向您说明在您使用我们的网站、服务及游戏时，我们如何处理您的个人资料。

1. 我们收集的信息

  • 账号信息： 我们收集您的用户名和密码的哈希（加密）版本。我们不以明文形式存储密码。
  • 游戏数据： 我们收集游戏分数和进度，以提供功能、排行榜并增强您的体验。
  • 付款数据： 付款处理由 {% if is_twa %}Google Play{% else %}PayPal{% endif %} 安全处理。我们不存储信用卡卡号。我们仅存储由 {% if is_twa %}Google{% else %}PayPal{% endif %} 提供的订阅 ID 和付款状态，以管理您的访问权限。
  • AI 隐私： 我们不会将您的个人数据、用户名或分数提供给外部人工智能模型进行训练。

2. 我们如何使用信息

  • 管理您的账号并提供高级方案的访问权限。
  • 通过 {% if is_twa %}Google Play{% else %}PayPal{% endif %} 安全地处理每月订阅。
  • 在排行榜上显示分数并存储您的学习进度。
  • 确保我们平台的安全性和稳定性。

3. Cookie 与安全性

我们使用对于安全登录、维持会话以及确保 {% if is_twa %}付款{% else %}PayPal{% endif %} 交易完整性至关重要的 Cookie。

4. 数据共享与第三方

我们仅与运营所需的信任伙伴共享数据：

  • {% if is_twa %}Google Play{% else %}PayPal{% endif %}： 用于付款处理和订阅管理。
  • Cloudinary： 用于提供优化的教育图像。
  • Google Analytics： 用于匿名使用情况分析。

5. 用户权利与数据删除

您有权随时访问、更正或删除您的数据。您可以通过“账户设置”页面立即永久删除您的账户及所有相关历史记录。一旦删除，数据将无法恢复。

6. AI 技术的使用

我们利用 AI 技术生成某些图像和教学内容。虽然我们追求高准确性，但这些内容仅供学习辅助之用，可能存在 AI 技术固有的局限性。

7. 国际数据传输

通过使用我们的服务，您同意将您的个人信息传输并在位于日本或美国等地的服务器上进行处理。我们将采取符合相关法律法规的安全保护措施，确保您的个人信息在跨境传输过程中的安全性。

本网站受 reCAPTCHA 保护，并适用 Google 的隐私政策和服务条款。我们使用 Google reCAPTCHA 来防止注册和登录表单受到垃圾邮件和自动软件的滥用。

10. 账号注销

用户登录后，可随时通过“账号设置”页面申请注销账号及相关数据。此外，用户也可以通过咨询表单联系我们申请注销账号。收到申请后，我们将永久删除您的数据。
